# ChargeHub AI Development Rules

## Project

ChargeHub is a cross-platform EV charging discovery mobile application.

Platform:
- React Native
- Expo
- TypeScript

Backend:
- Supabase

Goal:
Build a production-quality MVP.

---

## Rules

- Never generate unnecessary code.
- Keep components reusable.
- Follow atomic design.
- Use TypeScript everywhere.
- No inline styles.
- Use NativeWind.
- Use Zustand for state management.
- Use TanStack Query for API requests.
- Use React Navigation.
- Keep screens under 250 lines.
- Keep components under 150 lines.
- Prefer composition over duplication.
